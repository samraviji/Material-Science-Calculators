# Material Science Calculators

A free one-page materials science toolkit for students, researchers, and engineers.

## Live Features

- Hume-Rothery compatibility calculator
- Hall-Petch strengthening calculator
- Atomic % to Weight %
- Weight % to Atomic %
- Density calculator
- Diffusion / Arrhenius calculator
- Ternary alloy calculator
- Crystal structure reference
